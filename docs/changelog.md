```
統合前: ルートとinfohiroki/の重複構造
```

### 🔧 技術仕様

#### フロントエンド
- **HTML5**: セマンティックマークアップ
- **CSS3**: カスタムプロパティ、Grid、Flexbox
- **Vanilla JavaScript**: フレームワーク不使用

#### レスポンシブ
- **Mobile**: 768px以下（ハンバーガーメニュー）
- **Tablet**: 768px〜1024px
- **Desktop**: 1024px以上

#### パフォーマンス
- **GPU加速**: transform: translateZ(0)
- **Layout最適化**: contain: layout style
- **Debounce検索**: 300ms遅延
