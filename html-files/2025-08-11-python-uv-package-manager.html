🐍 Pythonの超高速パッケージ管理ツール「uv」入門

解説：Python VTuber サプーさん

📺 ソース動画はこちら

uvとは？ 🧩

uvは、Pythonの仮想環境構築とパッケージ管理を一つで行える、Rust製の非常に高速なツールです。従来のツールの長所を取り入れ、弱点を克服した次世代のパッケージ管理ツールとして注目されています。

  • 仮想環境とは？ 💻
    • プロジェクトごとに独立したPython実行環境を作成する仕組みです。
    • これにより、プロジェクト間でライブラリのバージョンが衝突するのを防ぎます。（例: プロジェクトAはDjango 3.0、プロジェクトBはDjango 4.0を使用）
  • パッケージの依存関係とは？ 🔗
    • あるパッケージを利用するために、別のパッケージが必要になる関係性のことです。（例: Djangoを使うにはsqlparseが必要）
    • パッケージ管理ツールは、これらの複雑な依存関係とバージョンを自動で解決してインストールしてくれます。

uvの3大特徴 ✨

  • 🚀 圧倒的な高速性
    • Rustで開発されており、処理が非常に速いです。
    • パッケージのダウンロードや依存関係の解決を並列処理することで高速化を実現しています。
  • 🧩 オールインワン管理
    • Python本体のバージョン管理、仮想環境の構築、パッケージ管理をuv一つで完結できます。
    • pyenv + venv + pip や Poetry のように複数のツールを組み合わせる必要がありません。
  • 🔄 高い再現性
    • pyproject.tomlとuv.lockファイルを使って環境を定義します。
    • これにより、チーム開発などで他のメンバーがuv syncコマンド一つで全く同じ環境を素早く再現できます。

⚠️ 注意点: 動画撮影時点（最新バージョン0.8）では、まだメジャーバージョンが1に達していません。今後、破壊的な仕様変更が行われる可能性がある点を理解して使用する必要があります。

基本コマンド一覧 🛠️

  • プロジェクト初期化
    • uv init <プロジェクト名>: 新規プロジェクトを作成し、設定ファイル（pyproject.tomlなど）を生成します。
  • 仮想環境の作成
    • uv venv: プロジェクト内に仮想環境（.venvフォルダ）を作成します。
  • パッケージ管理
    • uv add <パッケージ名>: パッケージを仮想環境にインストールし、pyproject.tomlとuv.lockを更新します。
    • uv add --dev <パッケージ名>: 開発時のみ使用するパッケージ（例: black, pytest）をインストールします。
    • uv remove <パッケージ名>: パッケージをアンインストールします。
  • 環境の同期・再現
    • uv sync: uv.lockファイルに基づいて、環境に必要なパッケージをインストール・同期します。
    • uv sync --upgrade: pyproject.tomlで指定されたバージョン範囲内でパッケージを最新版に更新します。
  • コマンド実行
    • uv run <コマンド>: 仮想環境内でコマンド（例: python main.py, pytest）を実行します。
    • uvx <パッケージ名> <引数>: 仮想環境を汚さずに、一時的にパッケージをインストールしてコマンドを実行します。
  • pip互換コマンド
    • uv pip list: インストール済みパッケージの一覧を表示します。
    • uv pip install -r requirements.txt: requirements.txtからパッケージをインストールします。（※この方法はpyproject.toml等を更新しないため注意）

まとめ

Pythonのパッケージ管理ツール「uv」は、Rust製で非常に高速な動作が特徴です。従来のツールが担っていた仮想環境の構築、パッケージのインストール、依存関係の解決、さらにはPython本体のバージョン管理までを一つのツールで完結させることができます。

プロジェクトの設定はpyproject.tomlファイルに記述し、依存関係を含む全パッケージの正確なバージョン情報はuv.lockファイルに記録されます。これにより、チーム開発時にuv syncコマンド一つで誰でも同じ開発環境を素早く再現でき、高い生産性を実現します。uvxを使えば仮想環境を汚さずにツールを一時実行することも可能です。高速性と利便性を両立した、注目の次世代ツールです。
