# Marpの特徴



- **Visual Studio Code**上で、Markdownからプレゼンスライドを生成
- **Marp**を使用して、デザインを自由自在にカスタマイズ
- 図表やグラフの挿入も可能

---

# Marpとは

- **M**arkdown **P**resentation **E**cosystem
- Markdownからプレゼンスライドを生成するツール
- VS Codeと組み合わせて使用可能

![width:400px](https://source.unsplash.com/random/600x400?markdown)


---

# Marpの利点

1. 一定品質のスライドがすぐできる
2. 普段使いのエディタで作成可能
3. 既存文書が再利用できる
4. デザインとコンテンツを分離
5. 調整が容易
6. Gitなどでバージョン管理が可能

## 2. Marp for VS Codeプラグインを追加

# 基本的な使い方

1. Markdownファイルを作成
2. `marp: true`を先頭に記述
3. `---`で各スライドを区切る
4. プレビューボタンでスライドを確認
5. エクスポートしてPDF等を作成

---

```markdown
---
marp: true
---

# スライド1

---

# スライド2
```

---

# スタイルの適用

## 全スライドに適用（Global directives）

```markdown
---
marp: true
theme: gaia
header: "Header text"
footer: "Footer text"
---
```


---

## 一部スライドに適用（Local directives）

```markdown
---
<!--
_backgroundColor: black
_color: white
-->

# タイトル
```

---

# 画像の扱い方

- 背景画像として使用: `![bg](image.jpg)`
- 位置指定: `![bg left:40%](image.jpg)`
- 明るさ調整: `![bg brightness:0.6](image.jpg)`

例:
```markdown
![bg left:40% brightness:0.7](image.jpg)

# 画像付きスライド
- 左側に背景画像
- 明るさを70%に調整
```

---

# カスタムテーマの作成

1. CSSファイルを作成
2. VS Codeの設定にテーマを追加
3. Markdownファイルでテーマを指定
---

```css
/* @theme custom */
@import 'default';

section {
    font-family: Arial, sans-serif;
}

h1 {
    color: #0066cc;
}
```

VS Code設定:
```json
"markdown.marp.themes": [
    "./path/to/custom-theme.css"
]
```
