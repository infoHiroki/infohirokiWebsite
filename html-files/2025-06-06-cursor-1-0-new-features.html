🎉 Cursor 1.0 の主要新機能

AIコードエディターの革新的アップデート

🚀 Cursor 1.0 正式リリース！

AI駆動のコードエディターCursorが1.0に到達。開発者体験を根本から変える新機能が満載です。

📋 公式変更ログ:

Changelog - Jun 4, 2025 | Cursor - The AI Code Editor

主要新機能

🤖

BugBot

自動コードレビュー
PRを自動的にレビューして、潜在的なバグや問題をキャッチする革新的な機能です。
  • GitHubのPRに自動でコメント
  • 「Fix in Cursor」ボタンで直接修正
  • 潜在的なバグの早期発見
  • コードクオリティの向上
⚡

Background Agent

全ユーザー展開
リモートコーディングエージェントが全ユーザーに利用可能になりました。
  • チャットのクラウドアイコンで起動
  • Cmd/Ctrl+Eのショートカット対応
  • バックグラウンドでの自動処理
  • パフォーマンスの大幅向上
📊

Jupyter Notebook

完全サポート
Jupyter Notebookでの複数セル作成・編集が可能になり、データサイエンス作業が大幅改善。
  • 複数セルの同時編集
  • 研究タスクの効率化
  • データサイエンス対応強化
  • シームレスな統合
🧠

Memories

記憶機能
会話から事実を記憶し、将来の会話で参照可能な革新的なメモリ機能。
  • 会話内容の自動記憶
  • プロジェクト単位での管理
  • コンテキストの継続性
  • より自然な対話体験
🔧

MCP ワンクリック

簡単インストール
MCPサーバーをワンクリックで設定可能。OAuth認証もサポート。
  • ワンクリックセットアップ
  • OAuth認証サポート
  • 設定の簡素化
  • 拡張性の向上

🚀 その他の改善点

  • チャット内でMermaid図やMarkdownテーブルの表示
  • PDFの解析とコンテキスト化
  • 並列ツール呼び出しによる高速レスポンス
  • 新しいダッシュボードと設定画面
  • ユーザーインターフェースの全面刷新
  • エラーハンドリングの強化
  • パフォーマンスの最適化
